Aprenda a Traduzir
Programas para cada função

Hexpose: Editor Hexadecimal, é um programa de DOS muito útil, eu considero o melhor Editor Hex para aprender a traduzir, já que ele possui uma busca relativa muito útil para encontrar o caracteres do TBL.

Hex Work Shop: Editor Hexadecimal para Windows, muito útil para quem usa um tradutor de inglês/português como o Globalink Power Translator Pro. Pois pode-se copiar as frases do editor e colá-las no tradutor.

Tile Layer: Editor de Tiles para DOS. Este programa edita as Tiles(imagens), das ROMs.

NAGAv18: Editor de Tiles para DOS. Antes de começar a editar as Tiles, ele mostra uma descrição da ROM. O que mais diferencia ele do Tile Layer é a possibilidade de modificar o tipo de leitura das Tiles.
