IPS - Definição - Aplicação - Criação

Definição

   IPS é um arquivo que é capaz de modificar as características de um rom, ou de qualquer tipo de arquivo. Foi inventado principalmente pra auxiliar os tradutores de roms. Pois com um IPS, não é preciso fazer novamente o download do rom traduzido, basta pegar só o IPS e aplicá-lo.

Aplicando

   Para aplicar um IPS, você necessita de um programa, neste tutorial, nós usaremos o SnesTool como padrão. Para aplicar, abra o programa e vá em "Usar IPS", selecione a IPS e aperte <enter>, depois selecione a ROM a ser aplicada o IPS. Para facilitar, copie o SnesTool para a pasta onde estão a ROM e o IPS.

Criando

   Após fazer uma tradução, você deve criar um IPS para facilitar o download dos usuários, pois o IPS é muito menor que o jogo. Para criar um IPS, você deve abrir o programa SnesTool, ir em "Gerar IPS", selecionar a ROM original e depois a traduzida. O IPS será criado na pasta onde se localiza o SnesTool.

Obs.: Para ficar mais fácil, pegue a versão traduzida do SnesTool, pela CBT. Snestool é encotrado na seção Utilitários.
